# Regional-Sales-and-Revenue-Analysis
Sales data analysis project using Excel, SQL, and Power BI. Cleaned and transformed dataset, handled missing values, and removed duplicates. Performed EDA to identify revenue trends, regional performance, and top products. Built interactive dashboard to visualize insights for data-driven business decisions.
# 📊 Sales Data Analysis and Business Insights

## 📌 Project Overview
This project focuses on analyzing a sales dataset containing Order ID, Product, Region, Revenue, and Date. The goal is to clean the data, perform analysis, and extract meaningful business insights using Excel, SQL, and Power BI.

---

## 🧹 Data Cleaning (Excel)
- Created a separate cleaned dataset to preserve original data
- Handled missing values using:
  - Mode for categorical columns (Region)
  - Mean for numerical column (Revenue)
- Standardized Product names and Date format
- Removed duplicate records
- Ensured data consistency and accuracy

---

## 📊 Data Analysis (Excel)
- Calculated total revenue
- Analyzed region-wise sales distribution
- Identified top-performing products

### Key Results:
- Total Sales: **36,522,968.87**
- Top Product: **Tablet**
- Highest Revenue Region: **South**

---

## 🗄️ SQL Analysis
Performed using MySQL Workbench:

- Total Revenue Calculation
- Region-wise Revenue Analysis
- Top 5 Products by Revenue
- Average Revenue Analysis
- Filtering high-value transactions

---

## 📈 Dashboard (Power BI)
An interactive dashboard was created with:
- Total Revenue KPI Card
- Region-wise Sales Bar Chart
- Product Performance Visualization
- Sales Trend Line Chart
- Slicers for Region and Product filtering

---

## 🔍 Key Insights
- South region generates the highest revenue
- Tablet is the top-performing product
- A few products contribute most of the revenue
- Sales vary significantly across regions

---

## 🚀 Tools Used
- Excel (Data Cleaning & Analysis)
- SQL (MySQL Workbench)
- Power BI (Dashboard Visualization)

---

## 📌 Conclusion
This project demonstrates how raw sales data can be transformed into meaningful business insights using data cleaning, analysis, SQL querying, and visualization techniques to support better decision-making.
